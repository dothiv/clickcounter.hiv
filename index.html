Click Counters

To encourage users to continuously visit .HIV domains and inform the about the progress of the current campaign we will show them a visual feedback every time they visit one of our domains.

We call this feedback click counters. This also serves as a tracking tool as we cannot tap directly into the DNS to track .HIV visits right now. We kindly ask every domain owner to integrate a one-line code snippet into their existing website or let their .HIV domain server a simple pre-generated HTML document in order to display the click counters.

Integration

As of now dotHIV will not be able to run DNS servers for .hiv domains, so we depend on .hiv domain owners to report their visits back to us.

We have three options for that:

Include a call to our banner script (preferred)

You modify your existing page and integrate a JavaScript tag which loads our banner from Google AppEngine. We use this to count visits.

Just add this line before the </body> tag of your site:

<script src="//dothiv-registry.appspot.com/static/banner.min.js" type="text/javascript"></script>

This page uses this approach.

Wrap your regular site in an iframe

If you do not want to modify your existing site, let your .hiv domain point to a separate webspace account containing a simple index.html that loads your regular page in an iframe. We can then overlay your regular site with our banner.

We have a demo of this technique, too.

Build your own counter

Our counter has a simple REST api which can be used to develop custom counters.

This can be used to report visitors to us.

Styles

They come in three different sizes. We have a large center version which is displayed whenever a user visits a .HIV domain for the first time. It's a real beauty, isn't it? As we don't want to get in the way on the next visits to that domain users will see either the top or right version which are much smaller and more compact. Where the center version is the high-five these are the small nods. Even if we could count visits without the click counters we'd like to show our appreciation every time …

Try them: center , top, right.

Technology

They are implemented in HTML and JavaScript, their source code is available on GitHub.

The backend for counting and delivering is implemented in Python and runs on Google AppEngine.

The configuration wizard on our platform is build on AngularJS and Symfony2.
