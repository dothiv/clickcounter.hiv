Der gute Klick

Mit jedem Klick auf eine .hiv-Seite soll der User eine Mikrospende auslösen. Und darauf Feedback in Form eines „Dankeschön“ bekommen. Wir nennen dieses Feedbacktool „Click-Counter“.

Auf dieser Seite erfahren Sie mehr zu diesem Tool und dessen Konfiguration.

Der Click-Counter

Das Tool zum Zählen der Klicks ist in jeder .hiv Adresse inbegriffen.

In der Standardversion erscheint er im dotHIV Design und gibt dem Besucher Feedback über die ausgelösten Spenden. Zusätzlich bedankt er sich recht herzlich für den Besuch. Er schließt sich automatisch nach 3 Sekunden. Der Banner ist in mehreren Sprachen und unterschiedlichen Größen verfügbar:

Die große Frage:
Weiterleitung oder auf .hiv-Seite bleiben?

Welche Webseite nach der Eingabe ihrer .hiv-Domain hinter dem Click-Counter angezeigt wird, entscheiden Sie:

Wenn Sie sich für die Variante „Weiterleitung“ entscheiden, erscheint hinter dem Counter die von Ihnen eingegebene Webseite. Also z.B. von website.hiv auf website.com. Das ist sinnvoll, wenn Sie Ihren Besuchern die Möglichkeit geben wollen, beim Besuch Ihrer Webpräsenz Gutes zu tun.

Wählen Sie „auf .hiv-Seite bleiben“ erscheint hinter dem Click-Couter der von Ihnen hinterlegte Inhalt Ihrer eigenständigen .hiv-Seite. Das ist sinnvoll, wenn Sie auf Ihrer .hiv-Webseite spezielle, themenbezogene Inhalte hinterlegt haben oder ihre Webpäsenz hauptsächlich unter Ihrer .hiv-Domain zu finden sein soll. Also z.B. bei symptome.hiv oder sport.hiv.

Der Standard Click-Counter ist kostenfrei.

Der technische Ablauf

Im Anschluss an die Domainregistrierung erhalten Sie eine Einladung zu Ihrem .hiv Profil. Dort müssen ein paar Konfigurationsschritte ausgeführt werden – der zeitliche Aufwand liegt bei ca. 5-10 Minuten)

Ergebnis ist ein Codeschnipsel, der ähnlich wie Google Analytics auf Ihrer .hiv Webseite hinterlegt  wird.

Fertig! Jeder Klick auf plus.hiv wird nun gezählt und unterstützt HIV-Projekte auf der ganzen Welt.

How it works in detail

As of now dotHIV will not be able to run DNS servers for .hiv domains, so we depend on .hiv domain owners to report their visits back to us.

We have three options for that:

Include a call to our click counter script (preferred)

You modify your existing page and integrate a JavaScript tag which loads our click counter from Google AppEngine. We use this to count visits.

Just add this line before the </body> tag of your site:

<script src="//dothiv-registry.appspot.com/static/clickcounter.min.js" type="text/javascript"></script>

This page uses this approach.

Wrap your regular site in an iframe

If you do not want to modify your existing site, let your .hiv domain point to a separate webspace account containing a simple index.html that loads your regular page in an iframe. We can then overlay your regular site with our click counter.

We have a demo of this technique, too.

Build your own counter

Our counter has a simple REST api which can be used to develop custom counters.

This can be used to report visitors to us.

Technology

The click-counters are implemented in HTML and JavaScript, their source code is available on GitHub.

The backend for counting and delivering is implemented in Python and runs on Google AppEngine.

The configuration wizard on our platform is build on AngularJS and Symfony2.
